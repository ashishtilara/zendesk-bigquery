# Zendesk to BigQuery

## Usage

1. clone the repo
2. copy `bin/environment.template.sh` to `bin/environment.sh`
3. fill values for environment variables in `environment.sh`
4. to run locally: `npm run local` or `npm start`
5. to deply: `npm run deploy`
